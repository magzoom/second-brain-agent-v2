🏆 Goal Tracker Diary

Каждый день в 09:00 legacy автоматически:

  1. Читает задачи, выполненные за последние 3 дня из Google Tasks
  2. Трансформирует их через Claude в достижения ("Позвонил врачу" → "Позаботился о здоровье")
  3. Публикует в канал Goal Tracker Diary с датой и категорией
✅ Дубликаты исключены — каждая задача публикуется один раз.

🔧 Решение проблем

ПроблемаРешение
Бот не отвечает launchctl list | grep com.sba.bot — должен быть PID.
Если нет: .venv/bin/sba service install bot
Apple Notes зависают / диалог «python3.12 would like to access data» System Settings → Privacy & Security → Automation → разрешить python3.12 доступ к Notes.
Также: Full Disk Access → включить оба python3.12 (symlink и реальный бинарь).
Google Drive: ошибка авторизации (invalid_grant) Запустить .venv/bin/sba auth google.
После авторизации inbox/fin_remind подхватят токен автоматически при следующем запуске.
Legacy нужно запустить вручную: CLAUDECODE="" .venv/bin/sba legacy или ждать 09:00.
Бот отвечает «что-то пошло не так» Написать /log в бот или проверить ~/.sba/logs/sba-bot.log
Inbox не обрабатывает файлы Проверить ~/.sba/PAUSED — если файл есть, удалить его
